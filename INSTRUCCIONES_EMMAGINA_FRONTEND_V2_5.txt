Emmagina Frontend v2.5

Cambios principales:
- Tarjetas de producto reducidas visualmente y redistribuidas en 65% imagen / 35% información.
- Imagen del producto ocupa todo el bloque superior de la tarjeta.
- Acciones recuperadas en tarjetas: Detalle, Comparar y Agregar.
- Comparación guarda hasta 3 productos en localStorage emmagina.compare.v1.
- CSS de tarjetas consolidado para evitar reglas duplicadas antiguas.

Subida recomendada:
1. Reemplazar el frontend completo por esta versión, manteniendo .git.
2. git add .
3. git commit -m "Emmagina frontend v2.5 tarjetas compactas"
4. git push
5. En Render, esperar deploy y recargar con Ctrl + Shift + R.
