¿Puedo enviar mi propio archivo?

Sí. Puedes enviar STL, 3MF, OBJ o una referencia para revisar si se puede fabricar.

¿Qué material usan inicialmente?

La base inicial es PLA, ideal para decoración, prototipos, accesorios y piezas de uso liviano.

¿Hacen retratos desde una foto?

Sí. La línea Crea tu Figura permite enviar una foto familiar, de pareja o mascota para evaluar una propuesta.
