EMMAGINA FRONTEND v3.3 - Editor del Sitio centralizado

Objetivo:
- Consolidar Banners, Contenido, Apariencia, Editor del Sitio y Modo Dios en un solo módulo visible: Editor del Sitio.
- Ocultar accesos duplicados del menú lateral admin.
- Permitir crear, editar, publicar, ocultar y eliminar páginas flexibles.
- Permitir editar bloques, imágenes Cloudinary, visibilidad, orden, márgenes y altura desde el Editor del Sitio.
- Mantener pagebuilder.html como redirección a editor-sitio.html por compatibilidad.

Rutas principales:
- /admin/editor-sitio.html
- /admin/pagebuilder.html redirige a /admin/editor-sitio.html
- /pagina.html?slug=mi-pagina para páginas nuevas creadas desde el editor

Validaciones:
- Inicio debe aparecer como página de sistema y no debe permitir eliminar.
- Nueva página debe crearse como borrador y aparecer de inmediato en la lista.
- Guardar página debe actualizar título, slug, publicación, navegación futura y SEO.
- Bloques deben poder agregarse, guardarse, moverse y eliminarse.
- Si subes una imagen, debes guardar el bloque para aplicar el cambio.

Subir:
git add .
git commit -m "Emmagina frontend v3.3 editor del sitio"
git push
