EMMAGINA FRONTEND V2.0 LIMPIO

Esta versión reconstruye el frontend público desde cero para eliminar duplicidades de la tienda anterior.

Público:
- index.html
- catalogo.html
- producto.html
- carrito.html
- pedido-personalizado.html
- páginas informativas básicas
- css/emmagina.css
- js/emmagina-*.js

Admin:
- Se mantiene /admin y sus dependencias porque ya está conectado al backend y permite administrar productos.
- El admin usa js/config.js para apuntar a https://emmagina-backend.onrender.com/api.

Cambios clave:
- Carruseles reconstruidos con transform, no scrollLeft.
- Autoplay, flechas y arrastre lateral con pointer events.
- Tarjetas de producto limpias, imagen cuadrada completa y datos abajo.
- Ficha producto funcional en producto.html?slug=... o producto.html?id=...
- Carrito local funcional con agregar primera opción/variante.
- Catálogo con filtros de categoría y búsqueda local.

Subida:
1. Reemplaza el contenido del repo emmagina-frontend por esta carpeta.
2. git add .
3. git commit -m "Emmagina frontend v2 limpio"
4. git push
5. Render hará auto-deploy si está en On Commit.
6. Recarga con Ctrl + Shift + R.
