Emmagina Frontend v4.0 - Render visual de bloques + navegación editable

Cambios principales:
- La Home y las páginas creadas desde Editor del Sitio renderizan secciones + bloques.
- Product Grid ahora se dibuja como grilla real, no como carrusel.
- Product Marquee se mantiene como carrusel horizontal continuo.
- El header carga navegación dinámica desde GET /api/pages/_navigation.
- Si una página creada está publicada y tiene "Mostrar en navegación" activo, aparece automáticamente en el menú.
- El Editor del Sitio agrega campo "Orden menú" para controlar posición.
- Mantiene fallback de navegación fija si el backend no responde.

Instalación:
git add .
git commit -m "Emmagina frontend v4.0 render navegacion editor sitio"
git push
