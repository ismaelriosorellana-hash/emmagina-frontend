Corrige la Home pública para renderizar bloques del CMS aunque el backend haya guardado tipos antiguos con guion medio:
- hero-banner o hero_banner
- info-cards o info_cards
- product-marquee o product_marquee

Instalación:
1. Reemplazar frontend completo sin borrar .git.
2. git add .
3. git commit -m "Emmagina frontend v3.7 render cms fix"

5. Probar con Ctrl + Shift + R.
