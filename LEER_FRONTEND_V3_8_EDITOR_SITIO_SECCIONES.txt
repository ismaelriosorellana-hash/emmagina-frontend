Emmagina Frontend v3.8 - Editor del Sitio con secciones

Cambios principales:
- El panel Editor del Sitio muestra páginas, secciones y bloques.
- Se puede crear más de una sección y varios bloques dentro de cada sección.
- Se agregó selector visual de tipos de bloque: ya no hay que escribir type manualmente.
- La lista lateral de categorías del Hero ahora es un bloque editable category_sidebar.
- La Home pública renderiza sections[] cuando existen, con compatibilidad para blocks[] antiguos.
