EMMAGINA FRONTEND v3.1 - Page Builder Admin

Qué incluye:
- Nueva página: /admin/pagebuilder.html
- Nuevo módulo JS: /admin/js/pagebuilder-admin.js
- Nuevo CSS: /admin/css/pagebuilder-admin.css
- Menú lateral admin actualizado con acceso "Modo Dios".

Cómo probar:
1. Reemplaza el frontend completo por este paquete.
2. Haz commit y push.
3. Abre https://emmagina-frontend.onrender.com/admin/login.html
4. Ingresa con tu usuario admin.
5. En el menú lateral abre "Modo Dios" o entra a:
   https://emmagina-frontend.onrender.com/admin/pagebuilder.html

Importante:
- Este panel usa las rutas del backend v1.8 Page Builder.
- Si la página home no existe, puedes crearla desde el panel o ejecutar en backend:
  npm run seed-pagebuilder-home
